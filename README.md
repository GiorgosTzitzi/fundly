# Fundly

Real Asset Capital Marketplace - Connecting project owners with investors in shipping and construction.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Project Structure

- `/app/onboarding` - Onboarding & KYC Flow with progressive disclosure
- `/app/marketplace` - Main Project Marketplace Dashboard with card-based layout
- `/app/project/[id]` - Project Detail & Interaction View with analytics

## Design Principles

- **Minimalist**: Apple-inspired clean design with 2-3 colors max
- **Card-based**: Scannable, responsive layouts
- **Progressive Disclosure**: Step-by-step onboarding to reduce cognitive load
- **Plain Language**: Friendly microcopy and clear instructions
- **High Contrast**: Accessible text and interactive elements
